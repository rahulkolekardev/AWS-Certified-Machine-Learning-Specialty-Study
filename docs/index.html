AWS Certified Machine Learning – Specialty Ebook

This offline ebook summarizes key AWS AI/ML services, data platforms, and supporting infrastructure for the AWS Certified Machine Learning – Specialty exam.

How to use this ebook for the exam

  • Start with the Table of Contents and skim each service page once, focusing on use cases, integration points, and exam tips.
  • Use the Cheat Sheet to compare similar services (for example Kinesis vs. Firehose vs. Managed Flink, or SageMaker vs. EC2).
  • Work through the practice questions to identify gaps, then jump back into the relevant service pages.
  • Keep the glossary handy for clarifying terminology in exam scenarios.
  • When in doubt, follow the official docs links on each page and read the “What is X?” and “Best practices” sections for deeper context.

Service coverage

The ebook covers data ingestion, storage and ETL, analytics, ML platforms, prebuilt AI services, compute, edge, monitoring, security, and BI tooling that commonly appear in the exam. Each service page includes:

  • Elevator summary and key features.
  • Typical ML pipeline use cases and integration points.
  • High-level architecture diagrams tailored to ML workloads.
  • Short AWS CLI and Python (boto3) examples.
  • Best practices for performance, reliability, and cost.
  • Exam tips and links to official AWS documentation and blogs.

About sources

Content in this ebook is paraphrased and summarized from official AWS documentation, AWS blogs, and re:Invent talks. Each service page links directly to its canonical docs. Due to the offline build environment, the summaries are based on the latest AWS documentation available to the model at training time rather than a live crawl.
