Exam-day cheat sheet

Ingestion

  • Streaming ingestion: Kinesis Data Streams (custom consumers, replay, low-latency).
  • Simple delivery into S3/Redshift/OpenSearch: Kinesis Data Firehose (managed, near real time).
  • Real-time stream processing: Managed Service for Apache Flink (stateful apps with Flink).

Storage & ETL

  • Durable data lake storage: Amazon S3 (versioning, lifecycle, storage classes).
  • Serverless ETL and catalog: AWS Glue (jobs, crawlers, Data Catalog).
  • Governed lake permissions: AWS Lake Formation (fine-grained access on lake tables).

Analytics & Query

  • Serverless SQL on S3: Amazon Athena (pay per query, integrates with Glue catalog).
  • Managed big data clusters: Amazon EMR (Spark, Hive, Presto, HBase).
  • Cloud data warehouse: Amazon Redshift (columnar MPP, Spectrum for S3).
  • Search & log analytics: Amazon OpenSearch Service.

Modeling platforms

  • End-to-end ML platform: Amazon SageMaker (studio, training, deployment, pipelines, feature store).
  • Generative AI foundation models: Amazon Bedrock (multi-model API, guardrails, agents).
  • AI assistants for builders: Amazon Q (code, business, and cloud assistants).
  • Ad-hoc training/inference: EC2 or Batch (more control, more ops).

Prebuilt AI APIs

  • NLP: Comprehend (classification, entities, key phrases, custom models).
  • Vision: Rekognition (labels, moderation, faces, text-in-image).
  • Speech to text: Transcribe (batch/streaming, custom vocabulary).
  • Text to speech: Polly (voices, neural TTS).
  • Translation: Translate (custom terminology, batch APIs).
  • Document understanding: Textract (forms, tables, queries).

Compute & orchestration

  • Flexible VMs: EC2 (GPU instances for training/inference).
  • Queue-based batch jobs: AWS Batch (spot integration, array jobs).
  • Containers registry: Amazon ECR.
  • Orchestration: ECS / EKS, with Fargate for serverless containers.
  • Edge ML: AWS IoT Greengrass for local inference and messaging.

Operations, security, and monitoring

  • Metrics and logs: CloudWatch (dashboards, alarms, logs).
  • Audit trails: CloudTrail (API history per account and region).
  • Network isolation: VPC (subnets, routing, endpoints for private S3/Bedrock/SageMaker).
  • Access control: IAM (fine-grained policies, roles, federation).
  • Visualization & dashboards: QuickSight (SPICE in-memory engine, ML insights).
