Amazon Rekognition

Amazon Rekognition is a computer vision service that can detect objects, scenes, faces, unsafe content, and text in images and videos. It provides pre-trained APIs and custom label detection for domain-specific use cases such as quality inspection or brand detection.

Key features

  • Image and video analysis for labels, moderation, faces, and text-in-image.
  • Face comparison and face search using face collections.
  • Custom Labels to train domain-specific detectors using labeled images in S3.
  • Integration with Kinesis Video Streams for near real-time video processing.
  • Support for bounding boxes and confidence scores for downstream logic.

Where it fits in ML pipelines

  • Adding vision capabilities to applications without training models from scratch.
  • Generating features from images or video frames for downstream ML or rules engines.
  • Moderating user-generated content for safety and compliance.
  • Building custom vision models via Rekognition Custom Labels instead of general-purpose frameworks.

How it works (high level)

CLI and SDK examples

Detect labels in an image (CLI)

aws rekognition detect-labels \
  --image \"S3Object={Bucket=ml-images-bucket,Name=example.jpg}\" \
  --max-labels 5

Detect labels (Python, boto3)

import boto3

rekognition = boto3.client("rekognition")

resp = rekognition.detect_labels(
    Image={"S3Object": {"Bucket": "ml-images-bucket", "Name": "example.jpg"}},
    MaxLabels=5,
)
for label in resp["Labels"]:
    print(label["Name"], label["Confidence"])

Best practices for ML workloads

  • Use Custom Labels for domain-specific tasks such as defect detection or brand identification.
  • Consider privacy and regulatory constraints for face recognition and biometric data.
  • Use confidence thresholds and multiple labels to reduce false positives in moderation flows.
  • Integrate with Kinesis Video Streams for scalable video analysis pipelines.
  • Cache or store results for frequently analyzed media to reduce costs.

Exam tips

  • Rekognition is the default answer for image/video understanding, moderation, and face-related use cases.
  • Custom Labels is important when questions mention custom vision categories without building models yourself.
  • For document text, Textract is used; for scene and object detection, Rekognition is correct.
