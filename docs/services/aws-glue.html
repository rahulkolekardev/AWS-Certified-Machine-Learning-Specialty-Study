AWS Glue

AWS Glue is a serverless data integration service that simplifies discovering, preparing, and moving data for analytics and ML. It combines a managed metadata catalog with serverless ETL jobs (Spark, Python shell) and visual data preparation tools.

Key features

  • Glue Data Catalog as a central schema and table registry for lakes.
  • Serverless ETL jobs running Apache Spark or Python.
  • Crawlers that infer schema and create/update table definitions automatically.
  • Glue Studio and DataBrew for visual job authoring and transformation.
  • Integration with Athena, Redshift Spectrum, EMR, and Lake Formation.

Where it fits in ML pipelines

  • Building curated feature datasets from raw S3 data using scheduled ETL jobs.
  • Maintaining a unified data catalog consumed by Athena and Redshift for exploration.
  • Preprocessing data before loading into SageMaker Feature Store or training jobs.
  • Transforming streaming outputs (from Kinesis or Flink) into partitioned lake tables.

How it works (high level)

CLI and SDK examples

Start a Glue job (CLI)

aws glue start-job-run \
  --job-name ml-curation-job

List tables in the Data Catalog (Python, boto3)

import boto3

glue = boto3.client("glue")

resp = glue.get_tables(DatabaseName="ml_lake")
for table in resp["TableList"]:
    print(table["Name"])

Best practices for ML workloads

  • Standardize on the Glue Data Catalog as the single source of truth for lake schemas.
  • Partition tables by date and other high-cardinality keys to improve query performance.
  • Use job bookmarks to process only new or changed data.
  • Separate raw, staged, and curated zones using distinct prefixes or buckets.
  • Integrate with Lake Formation for fine-grained table access permissions.

Exam tips

  • Glue appears whenever the exam describes “serverless ETL,” “data catalog,” or “schema discovery”.
  • Remember that Athena and Redshift Spectrum rely on the Glue Data Catalog for table metadata.
  • Lake Formation builds on the Glue catalog for governance; know their relationship.
